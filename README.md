# Gayrimenkul-Deger-Haritasi
Bu proje, İstanbul ili Çekmeköy ilçesinde yer alan gayrimenkullerin mekânsal dağılımını analiz etmeyi amaçlamaktadır. Çalışma kapsamında, bina bazlı rayiç bedeller ile sokak bazlı rayiç bedeller Coğrafi Bilgi Sistemleri (CBS) ortamında toplanmış, analiz edilmiş ve haritalandırılmıştır.
